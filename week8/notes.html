Translate Functions

Translation functions allow you to move elements left, right, up, or down.
These functions are simliar to position: relative;
When you employ a transition function, you're moving elements without impacting the flow of the document.
Unlike the postition: relative function, which allows you to posisition an element either against its current position
or against a parent or other ancestor, a translated element can only be moved relative to its current position.

Transitions

Transitions allow the values of CSS properties to change over time, essentially providing simple
animations. For example, if a link changes color on hover, you can have it gradually fade from one color
to the other transition can be gradual. If the browser lacks support for transitions, the change
will be immediate instead of gradual, which is fine accessible.
The steps to create a simple transition using CSS are as follows:
1. Declare the original state of the element in the default style declaration.
2. Declare the transition functions in your default style declaration using the transition
properties including: transition-property, transition-duration, transition-timing-function,
and transition-delay.
