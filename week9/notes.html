The Browser Object Model

The BOM is a collection of properties and methods that contain information about the browser and the computer screen.
For example, we can find out which browser is being used to view a page. We can also find out the dimensions of the screen
it is viewed on, and which pages have been visited before the current page. It can also be used for the rather dubious practice of creating
pop-up windows if you want to annoy your users. All the properties and methods within the BOM are made available
by the window object. Every browser window, tab, popup, frame and iframe has a window object.

How to find out what browser is being used using the window object:
window.navigaor.userAgent

How to find out location of the URL of the current page:
window.location.href
Some other location properties:
window.location.protocol : returns a string describing the protocol used (such as http, https, pop2, ftp )

window.location.host : returns a string describing the web domain

window.location.port : returns a string describing the port number.

window.location.pathname : returns a string of the path that follows the domain.

window.location.search : returns a string that starts with a '?' followed by a query string parameters.

Cookies

What are cookies? Cookies are small files that are saved locally on a user's computer. They were invented by Netscape
as a way of getting around HTTP being a stateless protocol. This means that a browser does not remember anything from one request to another.
So every time a user visits a page, nothing about any previous visits is remembered. Cookies can be used to sidestep this problem
by storing information that can then be retrieved between requests. A restriction of cookies is that they can only be read by a web page from the same
domain that set them. This is to stop sites from accessing information about users, such as other sites they have visited. Cookies are also limited
to storing up to 4KB of data, although 20 cookies are allowed per domain, which can add up to quite a bit.

Cookies are also used to personalize data to enhance their browsing experience, storing the users preferences, keeping track of their choices, (such as shopping)
authentication (purchases made, credit card info, name and address) and tracking users.

The following are creating/deleting cookie attributes using document objects:

Creating a Cookie:
document.cookie = "name=Johnny Bravo";
Creating a cookie, adding an expiration date:
document.cookie ="username=Johnny Bravo; expires=Thu, 25 Dec 2021 12:00:00 CST";
Creating a cookie, telling the browser what path the cookie belongs to:
document.cookie="username=Johnny Bravo; expires =Thu, 25 Dec 2021 12:00:00 CST;
path=/"
Delete a cookie:
document.cookie="username=; expires Wed, 01 Jan 2020 00:00:00 CST; path=/;"
Read a cookie:
document.cookie;
