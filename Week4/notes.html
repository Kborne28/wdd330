Forms

The form element helps receive input from the user, select menus, use buttons to submit information. Here's an example of a form that has one input field and a submit button to search:

Search

The form has a name attribute of search, and contains two controls, an input field where the user can search a phrase and a submit button to process the search.

You access a specific form by utilizing the document.form object. const form = document.forms.search
